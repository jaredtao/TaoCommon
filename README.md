# TaoCommon

收集一些常用的Qt后端代码，以便以后复用。

必须是经过实际项目验证过的。

# 内容

按文件夹分类

## Logger

  日志模板，来自[TaoLogger项目](https://github.com/jaredtao/taologger)

## Thread

  Qt多线程模板，来自[TaoThread项目](https://github.com/jaredtao/TaoThread)

  包括对Qt的Worker-Controller多线程模型的封装。

## Common

  一些通用的代码段。

  |文件名|功能|
  | ---- | -------------------------------- |
  |FileReadWrite.h|文件读写、Json读写|
  |ObjectMap.h|基础对象存储器;优先级对象存储器|
  |Singleton.hpp|单例模板|
  |Subject.hpp|观察者模板|
  |Common.h|QString 支持std::map; C++11 的 enum 和 int互转;快速计算md5;计算percent字符串;简易socket数据封包、拆包|
  |Tools|文件信息FileInfo相关处理;文件夹QDir相关操作。主要给Qml扩展|
